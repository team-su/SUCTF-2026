###  备注

>出题人可以在这里额外交待一些运维时的注意事项

在docker目录下构建容器

```bash
docker build -t su_who_is_the_thief .
```

启动容器

```bash
docker run -d -p 10002:8080 su_who_is_the_thief
```

默认是10002端口，如果需要修改端口号，只需要修改上述指令的10002即可
